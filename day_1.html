Penjelasan tentang git dan github
git init (berfungsi untuk menginisiasi git ke dalam Folder)
git add . (berfungsi mendaftarkan file-file yang ada di folder kita)
git commit -m "first commit" (berfungsi untuk menyimpan commit di dalam git)
git remote -v (berfungsi mengecek connect atau belum repositori lokal dengan jarak jauh) git push origin master (berfungsi untuk mengirim perubahan yang telah dibuat di cabang master lokal ke repositori jarak jauh yang ditunjuk oleh origin)
